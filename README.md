# Tic-Tac-Toe-project1
A console-based Tic Tac Toe game implemented using the C programming language.
This project focuses on implementing game logic using basic C concepts.

ABOUT THE PROJECT
------------------------------
This program allows two players to play Tic Tac Toe by entering positions on a 3Ã—3 grid.
It validates moves and determines the game result.

KEY CONCEPTS USED
---------------------------------
Arrays
Loops
Conditional statements
Functions

GAME RULES
-----------------------------------
Two players take turns
Each player marks one cell per turn
Three matching symbols in a row wins
Full board without a winner results in a draw
